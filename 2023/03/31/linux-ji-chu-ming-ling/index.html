linux基础命令

命令是可以嵌套使用的

ctrl+l 清空屏幕

ls

ls 展示当前目录下的文件

ls -l 以列表的形式将当前目录下的文件显示出来

ls -a显示所有文件包括隐藏文件

cd

直接用cd 命令会默认的回到我们的初始目录root权限下暨root，普通权限下暨home

cd +路径:切换到指定目录下

特殊路径符

cd .. 表示回到上一级目录

cd ../.. 返回上两级目录

cd ~ 表示会到home目录

cd - 表示返回上一个目录

pwd

pwd 打印当前目录

mkdir

mkdir+名称 创建一个以该名称的新目录

touch

touch + 文件名 创建一个该文件名的文件

cat

cat +路径 将内容全部显示出来

more

more + 文件路径,以翻页的形式打开文件

其中空格为下一页, 按q为退出

rm

rm + 文件 删除该目录下的文件

rm -r + 目录 删除该目录下的目录

rm -rf* 将该目录下的所有文件和目录删除

mv

mv aaa bbb 将aaa重命名为bbb

mv 路径a中x文件 路径b 表示将a路径的x文件剪切到b中

例如：mv /text2/text1/aaa /text2 就是将text1目录下的aaa文件剪切到text2中

cp

cp和mv的剪切是一个道理

cp 路径a中x文件 路径b 表示将a路径的x文件拷贝到b中

find

find + 目录+ 参数+ 文件名

例如：find /text2/text1 -name ‘a*’ 表示寻找text1目录下所有以a开头的目录。

打包压缩

tar -zcvf +打包压缩后的名称 + 想要打包压缩的文件

解压

tar -zxvf + 你想要解压的包名称

ps -ef 查看所有正在运行的进程

kill + pid(进程号） 强制杀死进程

ifconfig 查看网卡信息

ping 查看与某台机器连接的情况

netstat -an 查看当前系统端口

chkconfig iptables off 关闭防火墙
